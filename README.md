## â„¹ï¸ About The Project

**Shop Luvanski** is a Consumer-to-Consumer (C2C) e-commerce platform designed to empower small-scale vendors in the **South African informal market**.

The platform provides a secure, accessible, and user-friendly online marketplace supporting three distinct user roles: **Buyers**, **Sellers**, and **Administrators**. It aims to facilitate digital participation for informal traders, addressing local challenges like restricted digital access and trust concerns.

## ğŸš€ How to Use

* **Buyer Flow:** Register as a **Buyer** > Log in > Browse/Filter products > Add to Cart > Proceed through the 3-Step Checkout > Track order status on the **Orders** page.
* **Seller Flow:** Register as a **Seller** > Log in > **Create a Store** > Use **Manage Inventory** to list products > Fulfill and update status of orders via **Manage Orders**.
* **Admin Access:** Click **"Staff Access"** on the login page.
    * **Admin Username:** `admin`
    * **Admin Password:** `admin`
